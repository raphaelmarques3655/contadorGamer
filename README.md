# 🎮 Contador Gamer

Projeto desenvolvimento em React Native com Expo.

---

# Preview do APP 

O aplicativo possui:
* contador de pontos;
* botão de aumentar;
* botão de diminuir;
* botão de resetar;
* interface estilo gamer.

--- 

# tecnologias utilizadas

* React Native
* Expo 
* JavaScript

--- 

# como executar o projeto

## 1. clonar o repositório
```
git clone https://github.com/raphaelmarques3655/contadorGamer.git
```

## 2. Entrar na pasta do projeto
```
cd contadorGamer
```

## 3. Instalar depêndencias
```
npm install
```

## 4. Rodar o projeto
```
npx start
```
